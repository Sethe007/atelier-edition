Paramètres du projet
Fiche Œuvre · Personnages · Lieux · Configuration IA
Fiche Œuvre

Ces informations alimentent l'IA pour contextualiser ses analyses et suggestions.

Auteur & Page de garde

Ces informations apparaissent sur la page de garde du document exporté.

Fiches Personnages

Les noms et variantes sont transmis à l'IA pour ne jamais être signalés comme fautes.

Fiches Lieux

Les noms de lieux et leurs variantes sont exclus des corrections orthographiques.

Configuration IA

Sélectionnez votre fournisseur et entrez votre clé API pour activer le correcteur IA, l'analyse de style et le rapport éditorial.

Cliquez 💾 Enregistrer pour valider — les modules sont mémorisés par fournisseur.
💾 La clé API est sauvegardée avec le projet (.json). Protégez votre fichier projet.
⚡ Groq est gratuit sans CB — obtenez une clé sur console.groq.com
⇄ OpenRouter est gratuit sans CB — obtenez une clé sur openrouter.ai/keys
🛡 Le correcteur LanguageTool fonctionne toujours sans clé API.
Prompts IA

Personnalisez les instructions envoyées à l'IA pour chaque module. Les modifications sont sauvegardées avec le projet. Les valeurs par défaut sont toujours restaurables.

ℹ️ Le contexte de l'œuvre (fiche Œuvre, personnages, lieux) est injecté automatiquement — inutile de le réécrire dans les prompts.
✦ Utilisez {contexte_oeuvre} pour indiquer où il sera inséré si vous voulez le repositionner.
Préférences

Paramètres généraux de l'application. Ces réglages s'appliquent à toutes les sessions.

🌐 Interface
Langue de l'interface
Langue de tous les menus, boutons et messages de l'application.
Thème de l'interface
Couleur du header, de la barre latérale et des icônes.
✦ Intelligence artificielle
Langue de sortie IA
Langue dans laquelle l'IA rédige tous ses retours — corrections, analyses, rapports, propositions.
✐ Correction automatique
Activer la correction automatique
Corrections mécaniques légères en temps réel — sans réécriture, sans IA.
Majuscules automatiques
Majuscule après . ? ! et en début de paragraphe.
Supprimer les doubles espaces
Bonjour&nbsp;&nbsp;monde → Bonjour monde
Espaces de fin de ligne
Supprime les espaces traînants à la fin de chaque ligne.
Apostrophes typographiques
l'homme → l’homme
Guillemets typographiques
FR : « Bonjour » — EN : “Hello” — ES : «Hola»
Ponctuation normalisée
FR : Bonjour! → Bonjour ! — EN/ES : Hello ! → Hello!
Répétitions accidentelles
the the → the · de de → de (mots courts uniquement)
Ponctuation excessive
Bonjour..... → Bonjour…
Accord nominal (pluriel)
FR : les fleur → les fleurs · des chat → des chats
Accord sujet-verbe (pluriel)
FR : les chats mange → les chats mangent · EN : the cats runs → the cats run · ES : los gatos corre → los gatos corren
Accord nom-adjectif (pluriel)
FR : des résultats positif → des résultats positifs · EN/ES : accord adjectif-nom au pluriel
Orthographe lexicale
Corrige les mots mal orthographiés selon un dictionnaire FR/EN/ES (sans réécriture).
Participes passés (avoir)
FR : j'ai parle → j'ai parlé · il a manger → il a mangé
Les modifications sont appliquées immédiatement.
